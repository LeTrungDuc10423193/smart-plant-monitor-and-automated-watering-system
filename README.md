# smart-plant-monitor-and-automated-watering-system
IoT-based smart plant monitoring and automated watering system using Arduino Uno R3, Soil Moisture Sensor, DHT11, Relay Module, and Water Pump.

This project presents the development of a Smart Plant Monitor and Automated Watering System designed specifically for indoor plants. The system uses an Arduino Uno R3 together with environmental sensors to monitor soil moisture, room temperature, and humidity in real time. A mini water pump controlled through a relay module automatically irrigates the plant whenever the soil moisture falls below a predefined threshold. The system aims to reduce human error in plant care and support users with busy schedules or frequent travel.
